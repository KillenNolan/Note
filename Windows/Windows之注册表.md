# Windows之注册表(Regedit)

### 取消ftp中记住的账号密码

1. 首先在开始–运行中输入regedit，会显示一个注册表编辑器(Windows+R)
2. 其次，找到**FTP保存密码**的地方在HKEY_CURRENT_USER\Software\Microsoft\\**FTP**\Accounts\ 接着你可以看到你的网站域名
3. 最后，不想要的日志信息就可以把其相应目录下的内容删掉即可



### 查看电脑已连接的wifi密码

2. 选择连接wifi的状态-->无限属性-->安全-->显示字符,即可查看WIFI密码
